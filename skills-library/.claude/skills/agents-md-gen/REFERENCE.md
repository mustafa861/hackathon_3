# agents-md-gen Reference

## Customization

### Output Format
The generated AGENTS.md includes:
- Repository overview
- Detected tech stack
- Code conventions (auto-detected)
- Key files list (top 15)

### Supported Frameworks
- Next.js / React / TypeScript
- FastAPI / Python
- Prisma ORM
- Dapr microservices
- Kubernetes / Helm
- Docker

### Token Budget
Target: < 500 tokens for efficient AI context loading.

### Manual Overrides
After generation, you can manually edit AGENTS.md to add:
- Project-specific conventions
- Important architectural decisions
- API endpoint documentation
- Database schema summary
